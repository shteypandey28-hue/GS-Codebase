## 🎯 About

This is the official portfolio website for **Ganesh Soni** — a seasoned educator, AI evangelist, and career strategist with over 15 years of experience empowering institutions, professionals, and students across India.

The site showcases his core services:

| Service | Description |
|---------|-------------|
| 🎓 **Faculty Development** | Training programs for colleges & universities across Pune, Mumbai, and beyond |
| 🤖 **AI for ALL** | Making artificial intelligence accessible to educators, HR professionals, QA engineers & students |
| 🧭 **Career Guidance** | Resume writing masterclasses, interview prep, and career strategy sessions |
| 🧠 **Student Counselling** | Ikigai-based dream mapping and purpose discovery for school & college students |

---

## ✨ Features

- **🎨 Vibrant Dark-Mode Design** — 6-color accent palette (Flame, Magenta, Electric Blue, Violet, Lime, Amber)
- **🪟 Glassmorphism** — Frosted glass cards, badges, and overlays throughout
- **✨ Gradient Text** — Multi-color gradient headings that pop
- **🎭 Framer Motion Animations** — Smooth page transitions, staggered reveals, floating cards with bounce
- **📱 Fully Responsive** — Mobile-first with hamburger menu and adaptive layouts
- **🖼 Bento Grid Layout** — Modern asymmetric grid for the philosophy/about section
- **🏷 Color-Cycling Feedback** — Each institution feedback entry gets a unique vibrant accent color
- **🔄 Infinite Marquee** — Scrolling banner of partner institutions
- **🌈 Ambient Background Orbs** — Subtle colored radial gradients floating behind content
- **📊 Animated Stats** — Hero section counters with per-color accents
- **🔍 SEO Optimized** — Proper title tags, meta descriptions, semantic HTML

## 🚀 Getting Started

### Prerequisites

- **Node.js** ≥ 18
- **npm** ≥ 9

### Installation

```bash
# Clone the repository
git clone https://github.com/shteypandey28-hue/GS-Codebase.git
cd GS-Codebase

# Install dependencies
npm install

# Start the development server
npm run dev
```

The site will be running at **http://localhost:5173**

### Build for Production

```bash
npm run build
npm run preview
```
